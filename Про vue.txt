Чтобы store работала в файле index.js в папке routes, 

Это проблема практически всех сборщиков пакетов; предметы должны быть задекларированы, прежде чем их можно будет использовать.

Если А требуется Б, а Б требуется А во время импорта, есть циклическая зависимость, потому что одна будет загружена, требуя другую, которая еще не пришла.

В подобных случаях необходимо отложить (т.е. не запускать код в теле файла) любые зависимые взаимодействия до тех пор, пока обе стороны не загрузятся.

import router from '../router' или import VueRouter from "vue-router"; неточно!
import store from '../store'

// both files are imported now, so they can reference each other
router.beforeEach(function (...) {
  store.blah...
})
